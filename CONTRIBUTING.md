# Contributing

Thanks for contributing to NebulaWind.

## Development
- Install: `npm ci`
- Dev: `npm run dev`
- Quality gates:
  - `npm run check`
  - `npm run lint`
  - `npm run build`

## Guidelines
- Keep code clean-room (no copying from external templates/repositories).
- Prefer type-safe patterns.
- Write minimal tests when adding logic.
