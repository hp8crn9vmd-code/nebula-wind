# NebulaWind Docs

NebulaWind is a **clean-room**, production-first starter built from scratch.

## What you get
- Design tokens + theming foundation
- Type-safe blog (Astro Content Collections)
- RSS + Sitemap + robots.txt
- Local offline-ready search (Pagefind)
- Component library page
- CI quality gates (and Lighthouse in GitHub Actions)
